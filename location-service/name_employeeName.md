# Find Information of Faculties

```
GET /person/name/{facultyName}
```

## Description
> Find faculty's information by his/her **Chinese** name.

# Request
## Parameters
<table>
  <tr>
    <td><b>Parameter</b></td>
    <td><b><b>Required</b></b></td>
    <td><b>Description</b></td>
  </tr>
  <tr>
    <td><b>token</b></td>
    <td><i>yes</i></td>
    <td>Your API token</td>
  </tr>
</table>

## Example
```
GET /person/name/王小明
```

# Response

## Formats
- json
